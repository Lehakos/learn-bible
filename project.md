# 📱 Bible Verses Game - UI/UX Specification (Multi-Mode)

## 📦 Storage

- Использовать **IndexedDB**
- Хранить:
  - профиль пользователя
  - прогресс по стихам
  - пользовательские стихи (добавленные вручную)
  - достижения
  - открытые предметы
  - статистику по режимам

---

# 🎮 Игровые режимы

## 1. Определи ссылку

- Показывается текст стиха  
- Нужно выбрать:
  - книгу  
  - главу  
  - стих  

(например из [Псалтирь](chatgpt://generic-entity?number=0))

---

## 2. Найди текст

- Показывается ссылка  
- Нужно выбрать правильный текст стиха  

---

## 3. Собери стих

- Слова или части перемешаны  
- Нужно восстановить правильный порядок  

---

## 4. Заполни пропуски

- Текст с пропусками  
- Нужно вставить недостающие слова  

---

# 🧭 Основной пользовательский поток

1. Главный экран  
2. Выбор режима  
3. Игровая сессия  
4. Результаты  
5. Награды  
6. Возврат на главный экран  

---

# 🏠 1. Главный экран (Home)

## Цель

Показ прогресса и быстрый доступ к игре

## Элементы

- Аватар (текущий)
- Уровень
- XP прогресс-бар
- Кнопка "Играть"
- Навигация:
  - Режимы
  - Достижения
  - Аватар
  - Коллекция

## Поведение

- Загружает данные из IndexedDB
- Показывает уведомления:
  - новые достижения
  - новые предметы

---

# 🧭 2. Экран выбора режима

## Цель

Выбор типа активности

## Элементы

Карточки режимов:

- Определи ссылку
- Найди текст
- Собери стих
- Заполни пропуски

## Поведение

- При выборе:
  - создается игровая сессия
  - переход к игре

---

# 🎮 3. Игровой экран

## Общие элементы

- Индикатор прогресса (например 3/5)
- Индикатор серии (streak)
- XP за текущую сессию
- Кнопка выхода

---

## 3.1 Заполни пропуски

- Текст с пропусками
- Варианты слов или ввод

---

## 3.2 Собери стих

- Перемешанные слова/фразы
- Drag & drop или кнопки

---

## 3.3 Найди текст

- Ссылка
- 3-4 варианта текста

---

## 3.4 Определи ссылку

- Текст стиха
- Варианты:
  - книга
  - глава
  - стих

---

## Поведение (общее)

- После ответа:
  - мгновенный фидбек
  - подсветка:
    - правильно
    - ошибка
- Автопереход к следующему заданию

---

# 🧾 4. Фидбек за задание

## Элементы

- "Правильно!" / "Ошибка"
- Полученный XP
- Серия

## Поведение

- Показывается кратко (1-2 сек)
- Не прерывает поток игры

---

# 🏁 5. Экран завершения сессии

## Элементы

- Статистика:
  - правильные ответы
  - ошибки
- Полученный XP
- Новый уровень (если есть)
- Полученные награды:
  - достижения
  - предметы

## Поведение

- Анимация наград
- Кнопки:
  - Играть снова
  - На главный экран

---

# 🏆 6. Экран достижений

## Элементы

- Список достижений:
  - название
  - описание
  - прогресс
- Статус:
  - выполнено
  - в процессе

## Поведение

- При выполнении:
  - подсветка
  - отображение награды

---

# 🎭 7. Экран аватара

## Элементы

- Превью персонажа
- Категории:
  - фон
  - тело
  - аксессуары
- Список предметов

## Поведение

- Заблокированные:
  - отображаются с замком
- Разблокированные:
  - можно экипировать
- Изменения применяются сразу

---

# 📖 8. Экран коллекции стихов

## Элементы

- Список стихов
- Кнопка "Добавить стих"
- Для каждого:
  - текст
  - ссылка (например из [Псалтирь](chatgpt://generic-entity?number=1))
  - статус:
    - изучен
    - в процессе

## Поведение

- Стихи добавляются после игры
- Можно:
  - открыть
  - повторить в игре
  - добавить новый стих вручную

---

## 8.1 Добавление нового стиха в коллекцию

## Цель

Дать возможность родителям/пользователям добавлять свои стихи для изучения

## Элементы формы

- Поле "Текст стиха" (обязательное)
- Поле "Книга" (обязательное)
- Поле "Глава" (обязательное, число > 0)
- Поле "Стих" (обязательное, число > 0)
- Кнопки:
  - "Сохранить"
  - "Отмена"

## Поведение

- При сохранении:
  - выполняется проверка обязательных полей
  - проверяется дубликат по книге/главе/стиху
  - новый стих сохраняется в IndexedDB
  - новый стих сразу отображается в коллекции
- При ошибке валидации:
  - показывается понятное сообщение пользователю

- Новый уровень
- Достижение

## Поведение

- Показываются:

# 🔄 10. Общие UX принципы

## Быстрый старт

- 1-2 клика до игры

## Короткие сессии

- 2-5 минут

## Постоянный фидбек

- за каждый ответ
- за прогресс

## Видимый прогресс

- XP
- уровень
- открытия

## Разнообразие

- разные режимы
- постепенное открытие
